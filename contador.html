Contador de Vogais

Escreve uma palavra ou frase e vê, letra a letra, quantas vogais existem — atualizado ao vivo enquanto escreves.

0 vogais encontradas
Como funciona o código

O texto é passado para minúsculas e percorrido caractere a caractere com um ciclo for. Cada letra é testada com "aeiou".includes(letra) — se for vogal, o contador soma 1.

for (let i = 0; i < texto.length; i++) {
  if ("aeiou".includes(texto[i])) total++;
}
