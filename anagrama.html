Anagrama

Duas palavras são anagramas quando têm exatamente as mesmas letras, só que por outra ordem. Escreve duas palavras e descobre.

Como funciona o código

Cada palavra é passada para minúsculas, sem espaços, depois dividida em letras com split(""), ordenada alfabeticamente com sort() e reconstruída com join(""). Se as duas strings resultantes forem iguais, são anagramas.

a = a.toLowerCase().replace(/ /g, "").split("").sort().join("");
b = b.toLowerCase().replace(/ /g, "").split("").sort().join("");
a === b // true → anagramas
